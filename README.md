# SimpleObjectiveCApp
iPhone app - Ask the user for their information, and save the data entered by the user into an SQLite Database

1) Have a home page with a button to redirect to a page asking user information.
2) Ask the user for their name, address, phone number, email, age (slider 0-100), gender (switch) and date of birth (date), an option to to select an avatar
3) Have a submit button with an alert box thanking “name” of “email” for their time, and save the data entered by the user into an SQLite Database.
4) Have a new page with a table view that will display previously entered information. It will also display the avatar.
5) Add data saved outside of the database is to be saved in the App Delegate and accessed within the view controller.
6) Have a second button redirecting to an app page where it is about yourself.
7) Have a third button redirecting to a page with a web view opening a website.
